# Sito gratuito — Ristorante Le Regioni

Il sito è già pronto per essere pubblicato gratuitamente con GitHub Pages.

## 1. Inserisci i dati reali
Apri `config.js` e modifica:
- telefono;
- numero WhatsApp;
- indirizzo del ristorante;
- orari;
- link Google Maps.

## 2. Aggiungi il menù PDF
Quando il PDF sarà pronto:
1. Copialo nella cartella principale con nome `menu.pdf`.
2. In `index.html` cerca `PDF in preparazione`.
3. Sostituisci il collegamento `href="#"` con `href="menu.pdf"`.
4. Togli la classe `disabled` e cambia il testo in `Apri il menù`.

## 3. Inserisci fotografie vere
Sostituisci i riquadri nella sezione Galleria con immagini del locale, dei piatti e della famiglia.

## 4. Pubblicazione gratuita su GitHub Pages
1. Crea un account gratuito su GitHub.
2. Crea un nuovo repository chiamato `ristorante-le-regioni`.
3. Carica tutti i file e le cartelle di questo progetto.
4. Apri `Settings` → `Pages`.
5. In `Build and deployment`, scegli `Deploy from a branch`.
6. Seleziona `main` e cartella `/root`.
7. Salva. Dopo pochi minuti avrai un indirizzo gratuito simile a:
   `https://TUO-NOME.github.io/ristorante-le-regioni/`

## Nota
I numeri, l'indirizzo e gli orari presenti nelle vecchie immagini erano dimostrativi e non sono stati inseriti nel sito.


## Pagine del menù inserite
Il sito contiene ora nove pagine del menù in una galleria interattiva. Su telefono o computer basta toccare una pagina per ingrandirla e scorrere avanti o indietro.
